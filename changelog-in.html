▶ v3 build pengembangan

• Opsi baru:
- sebuah mod installer yang berfungsi
• Fitur baru:
- Sekarang lebih cepat dengan Java 8.
- Sekarang bisa meluncurkan Minecraft 1.13 dan seterusnya. Hoorah!
- Sekarang bisa meluncurkan Forge dan Fabric! Untuk meinstal salah satu dari mod ini, tinggal download saja file .jar mereka.
- Dukungan multi-arsitektur: ARMv7, aarch64, x86, dan x86_64.
- Sekarang mendukung control kustom (baik dibuat oleh komunitas Discord atau oleh anda!)

▶ v2.5.0 pratinjau (tidak digunakan)

• PERHATIAN: OptiFine Installer masih dalam percobaan, untuk beberapa perangkat akan crash di splash screen (screen Mojang). Jangan melapor bug tentang OptiFine crash di splash screen, mereka tidak akan diselesaikan.
• Pengaturan baru:
- Anda sekarang bisa mengganti MS delay dari gerakan sentuh (menempatkan block, menghancurkan block.)
• Fitur baru:
- Scrolling di Minecraft 1.8 atau seterusnya dengan menyentuh layar dan menyentuh terus SEC

▶ v2.4.2 pratinjau

• Berita baik: semua perangkat Android 7.0+ sekarang bisa menjalankan Minecraft secara normal.
• Perubahan:
- Ditambahkan OptiFine installer!
- Dukungan keyboard dan mouse external (Fitur membutuhkan Android 8.0+. Tidak dapat mendeteksi mouse event kiri atau kanan).
- UI Launcher sekarang akan sesuai di layar yang besar.
- Mode freeform sekarang sudah bisa.
- Mouse virtual sekarang sudah bisa.
- Mengganti multdoj7 dengan PojavDX (dx 1.16) untuk konverter lebih baik (terima kasih ke artdeell untuk menyarankan.
• Perbaikan:
- Touch event dipicu sebagai click event, bug ini sudah ada dari Boardwalk.
- (Dalam permainan) Mouse event dipicu setelah melihat sekitar.

▶ v2.4 BETA rilis

• Ingat: Jika perangkat anda menggunakan Android versi diatas atau sama dengan 7.0, gunakanlah aplikasi VMOS. VMOS akan menginstall Android 5.1 OS ke perangkat anda. Anda bisa mengambilnya untuk menjalankan aplikasi ini (VMOS telah dihapus oleh Play Store, anda harus menginstalnya secara manual)
• Perubahan:
- Penghapusan OpenJDK dan busybox.
- Ditambahkan kembali native library ARM 64bit.

• Perbaikan:
- Diperbaikan error yang tidak membolehkan user untuk menjalankan Minecraft tanpa koneksi internet.
