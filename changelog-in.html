Rilis Stabil

Dukungan launcher diterima di:


Hanya untuk Dahlia.

Dahlia

• Aplikasi sekarang membutuhkan sebuah akun premium, jika tidak, pemasangan aset permainan akan ditolak launcher sendiri.
• Sebagian besar bahasa-bahasa telah ditambah
• VGPU dan semua variasi-variasi GL4ES 1.1.5 telah dihapus dari launcher.
• GL4ES 1.1.4 telah diganti dengan sebuah fork kustom, menambah banyak fitur dan perbaikan bug
• Kontrol-kontrol telah dikerjakan ulang dan lebih baik dan lebih canggih, latensi gamepas sekarang frame perfect.
• Selektor warna untuk kontrol telah diganti
• Profil-profil telah ditambah! Sekarang anda bisa menetapkan beberapa setup permainan berbeda dengan seleksi renderer berbeda, JVM, dan direktori permainan.
• Java 17 sekarang diintegrasi ke launcher untuk menjalakan 1.17+

• Ada psi-opsi yang telah ditambah:
"Gunakan surface rendering alternate", yang menggunakan sebuah tipe berbeda surface rendering untuk merender permainan lebih cepat di beberapa perangkat.
"Mulai dengan mouse virtual menyala", yang membuat permainan meluncur dengan mouse sudah dinyalakan
"Nyalakan mode perfoma berkelanjutan", yang melimit penggunaan CPU/GPU untuk mencegah kepanasan dan thermal throttling

• Ditambahkan dukungan Arc Capes (sekarang diganti ke cosmetica.gg)

Perbaikan bug:
• Diperbaiki terjadinya crash saat menyalakan sebuah rekorder layar
• Diperbaiki sebuah crash NullPointerException jika meluncur versi-versi dibawah 1.6.4
• Di versi diatas 1.13, permainan bisa crash saat menggeser, ini telah diperbaiki.
• 1.13/1.14 sekarang tidak crash pada saat sentuhan pertama
• Diperbaiki versi-versi lama tidak berada di mode layar penuh saat peluncuran
• Diperbaiki versi-versi lama mempunyai "setengah cahaya" di entitas-entitas
• Diperbaiki hotbar tidak mendaftar input 0-9.
• Tidak akan lagi ada eror tentang null array saat memuat kontrol custom di Android 10+
• Permainan sekarang akan mengubah ukuran di splitscreen/mode freeform, meskipun tidak didukung secara resmi

Crocus (v3_openjdk)

• Opsi baru:
- Sebuah instaler mod yang berfungsi penuh
——————
• Fitur-fitur baru:
- Sekarang lebih cepat dengan Java 8.
- Sekarang bisa menjalakan Minecraft 1.13 dan lebih baru
- Sekarang bisa meluncurkan Forge dan Fabric dengan benar! Untuk menginstal salah satu dari "pemuat mod" ini, unduh saja berkas .jar mereka.
- Dukungan multi-architecture: ARMv7, aarch64, x86 dan x86_64
- Sekarang mendukung kontrol custom (bisa dibuat oleh komunitas Discord atau oleh anda!)
——————
• Apa yang baru
- Java 17 untuk menjalakan versi-versi Minecraft diatas 21w10a
- Revisi kontrol: penambahan fitur-fitur baru, seperti tombol drawer, dan fitur baru editing kontrol.
(Catatan: kontrol dibuat sebelum revisi ini tidak akan berfungsi.)
- Renderer baru: virgl dan vgpu
(Meskipun VirGL bisa mendukung sampai OpenGL 4.5, tetapi masih tidak bisa menjalankan shader dengan benar.)
- zink dihapus
- Implementasi akun lokal, setelah apa yang terjadi dengan Doregon/PojavLauncher_iOS, dan sekarang, anda tidak akan bisa mengunduh resource Minecraft, tidak satupun.
(Pastikan sudah membuat cadangan folder PojavLauncher anda supaya masih bisa tetap main Minecraft setelah pembaruan)


Rilis beta

Jangan memakai ini kecuali anda salah satu orang sejarah itu.


▶ v2.5 - Rilis beta teakhir

• Perhatian: Instaler OptiFine masih eksperemintal, untuk beberapa perangkat akan crash di layar splash (Layar Mojang). Jangan melapor bug tentang OptiFine crash di layar splash, mereka tidak akan diperbaiki.
• Opsi baru:
- Anda sekarang bisa memodifikasi delay MS dari gerak tangan (menaruh balok, menghancurkan balok.)
• Fitur baru:
- Menggeser di Minecraft 1.8 atau keatas dengan menyentuh layar dan menyentuh lama tombol SEC

▶ Rilis v2.4.2

• Berita baik: Semua perangkat Android 7.0+ sekarang bisa menjalankan Minecraft secara normal.
• Apa yang diganti:
- Penambahan instaler OptiFine!
- Dukungan mouse dan keyboard eksternal (Fitur membutuhkan Android 8.0+. Tidak dapat mengdeteksi event mouse kiri atau kanan).
- UI peluncur sekarang harusnya fit dengan layar yang besar.
- Mode freeform sekarang berfungsi.
- Mouse virtual sekarang berfungsi dengan benar.
- multdoj7 diganti oleh PojavDX (dx 1.16) untuk konverter yang lebih baik (terima kash ke artdeell untuk sarannya).
• Apa yang diperbaiki:
- Event sentuh dipicu sebagai event klik, bug ini sudah ada dari Boardwalk.
- (Di dalam permainan) Event mouse terpicu setelah melihat sekitar.


▶ Rilis v2.4

• Ingat: Jika perangkat anda mempunyai versi Android diatas atau sama dengan 7.0, gunakan aplikasi VMOS. VMOS akan menginstal OS Android 5.1 ke perangkat anda. Anda bisa mengambilnya untuk menjalankan aplikasi ini (VMOS telah dihapus dari Play Store, anda harus menginstalnya secara manual)
• Apa yang diganti:
- OpenJDK dan busybox dihapus.
- Native library ARM 64bit telah ditambah kembali.

• Apa yang diperbaiki:
- Diperbaiki sebuah eror yang mencegah pengguna dari meluncurkan Minecraft tanpa koneksi internet.

▶ Rilis v2.3.1

• Apa yang diganti:
- Ditambah seleksi teks di log pemasangan OpenJDK.
• Apa yang diperbaiki:
- Perangkat yang tidak bisa memasang OpenJDK tetapi bisa menyembunyikan erornya.

▶ Rilis v2.2/v2.3

• Apa yang diganti:
- Diganti beberapa kode di fungsi update - Penambahan busybox.
- Penambahan OpenJDK untuk konverter (Hanya untuk ARM 64bit).
- Penambahan banyak fungsi AWT.
- Screenshot (dan key F2) sekarang mengcapture langsung melalui API (javax.imageio).

▶Rilis v2.1

• Apa yang diganti:
- Penambahan tombol changelog!
- Versi custom sekarang bisa terlihat di list (tidak terimplementasi bagi pengguna)
- Pembaruan in-app sekarang berkerja.
- glshim digantikan oleh GL4ES untuk sebagian OpenGL 2.0 dan perfoma lebih baik.

▶ Rilis v2

• Apa yang diganti:
- [Sementara] Desain baru (copy dari original peluncur Minecraft 2017)
- Diganti cara pengguna dimuat/disimpan.
- Penambahan menu untuk seleksi opsi lebih banyak.
- Penambahan mouse virtual untuk styling mouse PC.

• Apa yang diperbaiki:
- Progres maksimal -1 saat mengkonvert.
- Verifikasi gagal saat menjalankan beberapa versi (dari patch berkas client)

Tentang PojavLauncher

PojavLauncher adalah sebuah peluncur Minecraft yang gratis, open source, dan stabil untuk Android dibuat dari Boardwalk, sebuah proyek peluncur MC dari Zhouwei Zhang
