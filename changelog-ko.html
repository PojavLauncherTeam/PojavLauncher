오직 달리아만

달리아

• 이제부터 프리미엄 계정이 필요합니다. 프리미엄 계정이 아니라면, 런처가 게임 구성요소를 설치하지 않을것입니다.
• 대부분의 언어가 추가되었습니다
• VGPU와 GL4ES 1.1.5의 모든 관련 버전들이 런처에서 삭제되었습니다.
• GL4ES 1.1.4가 버그 수정과, 많은 기능을 추가하며, 커스텀 포크로 대체되었습니다
• 컨트롤이 증가하고, 개선되었습니다. 컨트롤러 지원이 이제 완벽합니다
• 컨트롤에서 색깔 선택기가 수정되었습니다
• 프로필이 추가되었습니다! 이제부터 여러 다른 랜더러, JVM, 게임 디렉터리 구성이 유지됩니다.
• Java 17은 이제 1.17버전 이상 실행을 위해 런처에 내장되었습니다.

• 새로운 설정이 생겼습니다:
"대체 표면 렌더링 사용"은 특정 기기에서 게임을 더 빠르게 구동하기 위해, 다른 종류의 표면 렌더링을 사용합니다.
"가상 마우스 켜짐 상태로 시작하기"는 가상 마우스가 이미 켜진 상태로 게임을 시작합니다
"성능 유지모드 켜기"는 과열과 쓰로틀링 제어를 위해CPU/GPU의 성능을 제한합니다

• 아크 망토 지원을 추가했습니다 (이제 cosmetica.gg로 변경되었습니다)

버그 패치노트:
• 화면 녹화기를 실행할때 발생하는 크래쉬를 해결하였습니다
• 1.6.4 이하 버전을 실행할때 생기는 NullPointerException 크래쉬를 해결하였습니다
• 1.13 이상 버전에서 스크롤할때 생기는 크래쉬를 해결하였습니다
• 1.13/1.14 버전에서 처음으로 터치할때 생기는 크래쉬를해결하였습니다
• 오래된 버전에서 전체화면으로 시작되지 않는 문제를 해결하였습니다
• 오래된 버전에서 "엔티티 빛이 절반이 되는 현상"을 해결하였습니다
• 핫바에서 0-9 입력에 응답하지 않던 현상을 해결하였습니다
• 안드로이드 10 이상에서 커스텀 컨트롤을 로딩할때 생기던 "배열 없음" 현상을 해결하였습니다
• 비록 공식적으로 지원되지는 않지만, 이제부터 분할화면, 팝업화면 모드에서 게임이 자동으로 재조정됩니다

- 이제 원활하게 커스텀 컨트롤을 지원합니다 (디스코드 커뮤니티에서 만들어진것 또는 당신이 만든것도 지원합니다!)
——————
• 패치노트
- 마인크래프트 21w10a 이상 버전에서는 Java 17이 사용됩니다

(이번 업데이트 전에 만들어진 컨트롤 설정들은 사용할수 없음을 알립니다)

(VirGL이 OpenGL 4.5 이상을 지원하지만, 쉐이더를 원활히 실행할수 없습니다)
- zink를 삭제했습니다
- 로컬 계정을 조치하였습니다 Doregon/IOS용 PojavLauncher에 생긴 일처럼, 당신은 마인크래프트 구성요소를 단 한개도 설치할수 없습니다
(업데이트 후에도 마인크래프트를 플레이할수 있게, PojavLauncher 폴더를 백업해두세요)

▶ v2.5 - 최신 베타버전

• 주의: 옵티파인 설치기는 아직 테스트용 입니다. 어떤 기기는 스플래시 화면에서 멈출수 있습니다(모장 로딩화면). 옵티파인 충돌 관련 버그를 신고하지 말아주세요. 해결되지 않을 것입니다.
• 새로운 설정:
- 이제 터치 제스처 (블록 설치, 블록 파괴) 의 딜레이를 밀리초 단위로 설정할 수 있습니다.
• 새로운 기능:
- Minecraft 1.8 이상 버전에서 탭하고 SEC 버튼을 꾹 눌러 화면을 스크롤할 수 있습니다

▶ v2.4.2 미리보기

• 좋은 소식: Android 7.0 이상의 기기들을 사용하여 Minecraft를 정상적으로 실행할 수 있게 되었습니다.
• 바뀐 점:
- 옵티파인 설치기를 추가했습니다!
- 외부 마우스와 키보드를 지원합니다. (해당 기능은 Android 8.0에서 지원됩니다) 마우스 왼쪽과 오른쪽 이벤트를 감지할 수 없습니다).
- 이제 런처의 UI가 큰 화면에 딱 들어맞습니다.
- 이제 FreeForm 모드가 정상적으로 작동합니다.
- 가상 마우스가 이제 제대로 작동합니다.
- 더 나은 컨버터를 위해 multdoj7을 PojavDX (dx 1.16) 로 교체했습니다. (제안해주신 artdeell님에게 감사드립니다.).
• 고쳐진 것들:
- 클릭 이벤트로 트리거된 터치 이벤트. 이 버그는 보드워크에 존재했었습니다.
- (인게임) 마우스 이벤트가 주변을 둘러본 뒤에 트리거되었었습니다.


▶ v2.4 베타 릴리즈

• 기억하세요: 만약 Android 버전이 7.0보다 높거나 같다면, VMOS 애플리케이션을 사용하세요. VMOS는 Android 5.1을 기기에 설치할겁니다. You can take it to run this app (VMOS is removed by Play Store, you must install it manually)
