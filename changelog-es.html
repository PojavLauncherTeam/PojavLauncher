▶ Builds de desarrollo v3

• Nuevas opciones:
- Un instalador de mods completamente funcional
• Nuevas características:
- Ahora más rápido con Java 8.
- Ahora puedes ejecutar Minecraft 1.13 o superior. Hoorah!
- ¡Ahora puedes ejecutar Forge y Fabric correctamente! Para instalar uno de estos chicos malos, simplemente descarga sus archivos .jar.
- Soporte multi-arquitectura para: ARMv7, aarch64, x86 y x86_64.
- Ahora soporta controles personalizados (ya sea por la comunidad en Discord o por ti!)

▶ v2.5.0 preview (Versión obsoleta)

• NOTICIA: El instalador de OptiFine es experimental, es posible que para algunos dispositivos vaya a crashear en la pantalla de carga. No reportes bugs sobre OptiFine crasheando en la pantalla de carga, esto no va a ser solucionado.
• Nuevas opciones:
- Ahora puedes customizar los MS de los gestos táctiles(poner bloques, destruir bloques.)
• Nuevas características:
- Desplazarse en Minecraft 1.8 o superior, pulsando la pantalla y manteniendo pulsado el botón SEC

▶ v2.4.2 preview

• Buenas noticias: Cualquier dispositivo con Android 7 o superior puede ejecutar Minecraft normalmente.
• Cambios:
- Añadimos un instalador de OptiFine!
- Soporte para mouse y teclado externo (necesitas Android 8 o superior. Can't detect left or right mouse event).
- Launcher UI should now fit correctly in big screens.
- Freeform mode is now working.
- Virtual mouse now works properly.
- Replaced multdoj7 with PojavDX (dx 1.16) for better converter (thanks to artdeell for suggesting).
• Fixes:
- Touch event triggered as click event, this bug exist from Boardwalk.
- (In-Game) Mouse event triggered after looking around.
