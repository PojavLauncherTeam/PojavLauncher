▶ v3 开发版

• 新功能：
- 一个完整的 Mod 安装器
• 新特性：
- 现在在 Java 8 环境下可以更快的运行了
- 现在可以运行 Minecraft 1.13 及更高版本了 好耶！
- 现在可以正确地启动 Forge 和 Fabric了！ 想要安装这些玩意，只需要下载他们的 .jar 文件。
- 多构架支持：ARMv7, aarch64, x86 和 x86_64.
- 现在可以正确支持自定义控制了（由 Discord 社区或您创造）

▶ v2.5.0 预览版（已过时）

• 注意: OptiFine 安装器仍是预览，在一些设备上可能会在启动时崩溃（Mojang Screen）。 不要反馈任何关于 OptiFine 启动崩溃的问题，这些问题不会被修复。
• 新设置：
- 您现在可以自定义手势延迟长度（放置方块，破坏方块。）
• 新特性：
- 点击屏幕并按住 SEC 键可以在 Minecraft 1.8 及以上中滑动

▶ v2.4.2 预览版

• 好消息：现在所有 Android 7.0+ 设备都可以正常运行 Minecraft 了
• 更改：
- 新增了 OptiFne 安装器！
- 外置键鼠已支持（该功能需要 Android 8.0+ 。 目前无法检测左击和右击).
- 大屏设备上的启动 UI 现在可以正常显示了。
- 自由窗口模式现在可以正常工作了。
- 虚拟鼠标现在可以正常工作了。
- 将 multdoj7 替换为 PojavDX (dx 1.16) 以获得更好的转换（感谢 artdeell 提供建议）。
• 修复：
- 触摸事件会被当作点击事件，这个问题存在于 Boardwalk 。
- （游戏中）鼠标事件将会在移动视角时触发

▶ v2.4 BETA release

• 提醒：如果您的设备运行 Android 7.0 及以上，请使用 VMOS 应用。 VMOS 将安装 Android 5.1 OS 到您的设备。 您可以通过这个来运行该应用（VMOS 已经在 Play 商店下架，您需要手动安装它）
• 更改：
- 移除了 OpenJDK 和 busybox 。
- 重新加入了 ARM 64位 原生运行库。

• 修复：
- 修复了一个会导致用户在运行 Minecraft 时无法连接到网络的错误。
