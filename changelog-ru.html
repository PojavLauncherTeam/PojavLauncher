▶ Сборки для разработчиков в. 3

• Новые опции:
- полноценный установщик модификаций
• Новые функции:
- Теперь быстрее с Java 8.
- Теперь может запускать Minecraft 1.13 и выше. Ура!
- Теперь можно корректно запускать Forge и Fabric! Для установки просто загрузите JAR-установщики и запустите их в лаунчере.
- Поддержка архитектур: ARMv7, aarch64, x86 и x86_64.
- Теперь полностью поддерживается своё управлений (сделанное сообществом Discord или тобой!)

▶ v2.5.0 preview (устаревший)

• ЗАМЕТКА: Установщик OptiFine - экспериментальный, и на некоторых устройствах OptiFine будет останавливаться на экране с логотипом Mojang. Не отправляйте отчёты об ошибке с OptiFine аварийно завершившимся на экране запуска, они не будут исправлены.
• Новые функции:
- Теперь можно настраивать задержку для размещенияломания блоков в мс.
• Новые фишки:
- Скроллинг в Minecraft 1.8 и выше теперь делаеться нажатием на экран и удержанием ПКМ

▶ v2.4.2 preview

• Хорошие новости: Любое устройство с Android 7.0 и выше может запускать Minecraft без проблем.
• Изменения:
- Добавлен установщик OptiFine!
- Добавлена поддержка внешней клавиатуры и мыши (Для работы требуется Android 8.0+.) Не обнаруживает левый/правый клик).
- Теперь интерфейс лаунчера должен нормально отображаться на больших экранах.
- Режим свободного окна теперь работает корректно.
- Виртуальная мышь теперь работает корректно.
- multdoj7 заменён на PojavDX (dx 1.16) для лучшей конвертации (спасибо artdeell за предложение).
• Исправления:
- Нажатие на экран преобразуется в нажатии на мышь, эта проблема существует ещё с Boardwalk.
- (В игре) Обзор в игре вызывает нажатие мыши.

▶ v2.4 BETA release

• Запомните: Если версия Android на вашем устройстве равна или выше 7.0, используйте VMOS. VMOS установит ОС Android 5.1 на ваше устройство. Можете это использовать для запуска лаунчера (VMOS удалён из Play Store, нужно устанавливать вручную)
• Изменения:
- Удалено: OpenJDK и busybox.
- Возвращены нативные библиотеки для 64-бит ARM.

• Исправления:
- Исправлена ошибка не дающая пользователям запускать Minecraft без доступа к сети.
