Stabilní vydání

Podpora spuštěče přijata v:


Pouze Dahlia.

Dahlia

• Aplikace nyní vyžaduje prémiový účet, jinak instalace obsahu hry bude zamítnuta spouštěčem.
• Přidána většina jazyků
• VGPU a všechny varianty GL4ES 1.1.5 byly odstraněny ze spouštěče.
• Vlastní verze GL4ES byla nahradila GL4ES 1.1.4 přidávajíc mnoho funkcí a oprav chyb
• Ovládání bylo předěláno, vylepšeno a zvětšeno, latence gamepadu je perfektní.
• Výběr barev pro ovládání bylo změněno
• Profily přidány! Nyní můžeš míti více zachovaných sestavení hry s jinými vykreslovačemi, JVMy, a herními složkami.
• Java 17 je nyní integrována do spouštěče pro hraní 1.17+

Nová nastavení byla přidána:
»Použít alternativní vykreslování povrchu«, které používá odlišný typ vykreslování povrchu ke zrychlení hry na některých zařízeních.
»Začit se zapnutou virtuální myší«, které zapne myš při startu hry
»Zapnout režim trvalého výkonu«, které zlimituje CPU/GPU užití pro předcházení přehřívání a termálnímu zpomalování

• Přidána podpora Arc Capes (nyní změněna na cosmetica.gg)

Opravy Chyb:
• Opraveno spadnutí při spuštění nahrávání obrazovky
• Opraveno NullPointerException spadnutí při spouštění verzí pod 1.6.4
• Ve verzích nad 1.13, opraveno spadávání hry při skrolování.
• 1.13/1.14 již nepadají při dotyku
• Opraveno nebytí starších verzí na plnou obrazovku
• Opraveno »polosvětlo« na entitách ve starších verzích
• Opraven hotbar neregistrující vstupy 0-9.
• Již žádné chyby o nulovém řetezci při načítání vlastního ovládání na androidě 10+
• Nyní se hra bude škálovat v režimě rozdělených oken i režimě volných oken, i když ne oficiálně podporováno

• Nové funkce:
- Nyní rychlejší s Javou 8.

- Nyní může správně spustit Forge i Fabric! Pro nainstalování těchto "modloaderů" (načítačů modů), jen stáhni jejich jar soubory.

Vydání beta

Nepoužívej je, nejsi-li ten jeden historik.


▶ v2.5 - Poslední BETA

• POZNÁMKA: Instalatér Optifinu je experimentální, pro některá zařízení spadne na úvodní obrazovce (ta s Mojang načítáním). Don't report bug about OptiFine crash at splash screen, these will not be solved.
• New settings:
- You can now customize the MS delay of touch gestures (placing block, destroying block.)
• New features:
- Scrolling in Minecraft 1.8 or above by tapping the screen and holding SEC button

▶ v2.4.2 Release

• Good news: Any Android 7.0+ devices can now run Minecraft normally.
• Changes:
- Added OptiFine installer!
- External mouse and keyboard support (Feature needs Android 8.0+. Can't detect left or right mouse event).
- Launcher UI should now fit correctly in big screens.
- Freeform mode is now working.
- Virtual mouse now works properly.
- Replaced multdoj7 with PojavDX (dx 1.16) for better converter (thanks to artdeell for suggesting).
• Fixes:
- Touch event triggered as click event, this bug exist from Boardwalk.
- (In-Game) Mouse event triggered after looking around.


▶ v2.4 Release

• Remember: If your device has Android version above or equals to 7.0, use VMOS app. VMOS will install Android 5.1 OS to your device. You can take it to run this app (VMOS is removed by Play Store, you must install it manually)
• Changes:
- Removed OpenJDK and busybox.
- Re-added ARM 64bit native libraries.

• Fixes:
- Fixed an error that prevent users launch Minecraft without internet connection.

▶ v2.3.1 Release

• Changes:
- Added text selection on OpenJDK install log.
• Fixes:
- Devices can't install OpenJDK but able to hide the error.

▶ v2.2/v2.3 Release

• What changed:
- Changed some code in update function - Added busybox.
- Added OpenJDK for converter (ARM 64bit only).
- Added more AWT functions.
- Screenshot (and F2 key) now capture using directly API (javax.imageio).

▶v2.1 Release

• Changes:
- Added Changelog button!
- Custom versions now visible in the list (not implemented for user)
- In-app Update now usable.
- Replaced glshim with GL4ES for partial OpenGL 2.0 and better performance.

▶ v2 Release

• Changes:
- [Partial] New design (copy of 2017 original Minecraft Launcher)
- Changed how users load/save.
- Added Menu for more options to select.
- Added virtual mouse for styling PC mouse.

• Fixes:
- Max progress -1 when converting.
- Verification failed when run some versions (by patch the client file)
