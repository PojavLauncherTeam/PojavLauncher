PojavLauncher

Minecraft: Java Edition launcher for Android and iOS

Inherited from the first-ever launcher called 'Boardwalk', this is a Minecraft: Java Edition launcher for both Android and iOS

Faster and more Modern

Unlike Boardwalk (and mostly the old versions of PojavLauncher), the codes are coder-friendly. Plus, modern minecraft versions can run on a user-acceptable FPS. It also has a variety of renderer choices to choose from, including the highly-confidential so-called 'Zink' and glshimgl4es 1.1.5 that can almostly run 21w10a and above.
